# Issue Tracker

This repo tracks work in GitHub Issues:

- Repository: `Open-Smart-Irrigation/osi-os`
- Remote: `https://github.com/Open-Smart-Irrigation/osi-os.git`
- Preferred CLI: `gh`

Matt Pocock skills that publish work (`to-prd`, `to-issues`, `triage`) should use GitHub Issues by default. If GitHub auth is unavailable, draft the issue body locally and report the exact `gh issue create` command instead of inventing issue IDs.

For cross-repo Terra or cloud-backend work, check whether the implementation belongs in `osi-server` before creating an `osi-os` issue.
