# Triage Labels

Use the default Matt Pocock triage roles unless the GitHub repo has a more specific existing label:

- `needs-triage` - maintainer needs to evaluate
- `needs-info` - waiting on reporter
- `ready-for-agent` - fully specified and suitable for agent implementation
- `ready-for-human` - needs human implementation or product judgment
- `wontfix` - will not be actioned

Do not create new labels without checking existing repository labels first. If label creation is unavailable, include the intended labels in the issue body.
